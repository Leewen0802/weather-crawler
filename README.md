## Preface

練習 nodejs, puppeteer, travis-ci

### How it work
- 使用 puppeteer 抓取 www.cwb.gov.tw

- 資料將存為 `./data/json`
於 `./index.html` 透過 js 顯示~

- Update by travis daily Cron Jobs
